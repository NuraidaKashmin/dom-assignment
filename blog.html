What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a way of organizing a webpage by browsers. The page's HTML turns into a tree of objects like text, images, and buttons. Developers use JavaScript to change or update the page through the DOM and make it interactive without needing to reload the page.

How do you select an element from the DOM?

To select an element from the DOM, use JavaScript methods like getElementById(), getElementsByClassName(), or querySelector().

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element and handle events for its children. It improves performance by reducing the number of listeners. It also simplifies managing events, especially for dynamic or added elements.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes using the DOM, you can use methods like setAttribute() and getAttribute(). To change styles, use element.style.property.
